📈 Kingsley’s Math Progress

This page provides a snapshot of current learning goals, progress, and suggested practice focus. Updated weekly.

🎯 Current Learning Focus

Addition with Regrouping Place Value

Kingsley is currently working on understanding addition when numbers need to be regrouped (carried), along with strengthening place value (hundreds, tens, and ones).

✅ Skills Demonstrated

  • ✔ Identifying tens and ones accurately
  • ✔ Adding two‑digit numbers without regrouping
  • ✔ Explaining what regrouping means using blocks
  • ✔ Skip counting by 25s

🛠 Skills In Progress

  • ➕ Consistently carrying the extra ten in addition problems
  • 🔢 Identifying the correct value of digits in 3‑digit numbers

📅 Weekly Practice Expectations

Kingsley is encouraged to complete the 15‑minute daily practice sessions focused on:

  • Quick addition problems
  • Place value questions (tens and hundreds)

Consistent short practice sessions help build confidence and accuracy.

➡️ What’s Coming Next

Subtraction Foundations

Once addition and place value skills are secure, instruction will move into understanding subtraction as “take‑away” and comparison.

🏠 How Families Can Help

  • Ask Kingsley to explain how he solved a problem
  • Encourage him to check his work for carried numbers
  • Celebrate effort, not just correct answers
This progress page is for family reference and is updated regularly.
